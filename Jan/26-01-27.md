## 📝 오늘 배운 내용 요약

1. 네트워크 기초 용어
    - 시스템
        - 내부 규칙에 따라 능동적으로 동작하는 대상
    - 인터페이스
        - 시스템과 시스템을 연결하기 위한 표준화된 접근 방법
    - 전송매체
        - 시스템끼리 데이터를 전달하기 위한 물리적인 전송 수단
    - 프로토콜
        - 전송 매체를 통해 데이터를 교환하기 위한 특정 규칙
    - 네트워크
        - 프로토콜을 사용해 데이터를 교환하는 시스템의 집합을 통칭
    - 인터넷
        - 전세계의 네트워크가 유기적으로 연결되어 동작하는 통합 네트워크
2. 프로토콜
    - 프로토콜의 정의
        - 컴퓨터 네트워크를 하기 위해 송수신 장치 간 통일된 통신 규약
    - 종류
        - 독점적 프로토콜(SNA, IPX/SPX, AppleTalk 등)
        - 비독점적 프로토콜(TCP/IP 등)
            - Application
            - Host-to-host or transport
            - Internet
            - Network access
            - Physical
        - OSI 7 계층
            - 7 계층 : 응용 계층
            - 6 계층 : 프레젠테이션 계층
            - 5 계층 : 세션 계층
            - 4 계층 : 전송 계층
            - 3 계층 : 네트워크 계층
            - 2 계층 : 데이터 링크 계층
            - 1 계층 : 물리 계층
        - OSI모델에 5,6,7 계층은 [어플리케이션 레이어]라 부름.
3. Network Scale
    - PAN(개인 영역 네트워크)
        - 개인의 작업 공간 내에서 전자 기기들을 상호 연결(컴퓨터, 스마트폰, 태블릿 등)
    - LAN(로컬 영역 네트워크)
        - 거주지,건물과 같은 제한된 영역 내에서 컴퓨터를 상호 연결
    - WAN(광역 네트워크)
        - 넓은 지리적 영역에 걸쳐 확장되는 통신 네트워크
    - CAN(캠퍼스 네트워크)
        - 제한된 지리적 영역 내에서 여러 LAN을 상호 연결하여 구성
        - 캠퍼스 영역 네트워크는 LAN보다 크지만 규모는 작다
    - MAN(대도시 지역 네트워크)
        - 대도시권 크기의 지리적 영역에서 사용자와 컴퓨터 자원을 상호 연결
        - 도시 내 LAN을 하나의 더 큰 네트워크로 상호 연결하여 WAN에도 효율적으로 연결할 수 있는 것을 의미
