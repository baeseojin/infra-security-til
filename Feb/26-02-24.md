## 📝 오늘 배운 내용 요약

1. Apache
    - 정적 웹 서버 동작 방식
        1. 80 포트 연결
        2. 사용자 문서(index.html) 요청
        3. 서버가 시스템에서 웹 문서 검색 
        4. 서버가 요청한 파일을 읽음
        5. 서버가 클라이언트에게 웹 문서 전송
    - 동적 웹 서버 동작 방식
        1. 80 포트 연결
        2. php 프로그램 요청
        3. 서버가 php 프로그램을 실행시킨 후 클라이언트가 요청한 매개변수 넘겨줌 
        4. 서버가 php 프로그램으로부터 생성된 결과를 읽음
        5. php 프로그램으로부터의 결과를 웹 서버는 클라이언트의 브라우저에 전송
    - Apache 설정

    - Apache 주요 디렉터리
        - `/var/www/` 웹 페이지 콘텐츠 기본 디렉터리 위치
        - `/etc/httpd/conf/` 웹 서버 주 설정 파일인 `httpd.conf` 파일 위치
        - `/etc/httpd/conf.d/` 웹 서버의 추가 설정 파일 위치
        - `/etc/httpd/conf.modules.d/` 웹 서버와 함께 설치된 모듈 설정 관련 위치
        - `/usr/shard/httpd` 테스트 페이지, 에러 페이지 등 기본 콘텐츠 위치
        - `/usr/share/doc/httpd` 웹 서버 관련 문서 파일 위치
2. 가상 호스트
    - 한 시스템에서 여러 개의 도메인과 호스트 이름을 구분하여 웹 서비스 가능
    - 연결된 서버의 IP주소와 클라이언트가 요청한 호스트 이름의 조합에 따라 다양한 구성 설정 사용 가능
    - 이름 기반의 가상 호스팅
        - 네임서버가 각 호스트명이 올바른 IP 주소로 대응하도록 가상 호스트 설정
        - 하나의 IP 주소를 가지고 여러 호스트에 대해서 웹 서비스 제공
        - `/etc/httpd/conf.d/` 안에 파일 작성
        - `/etc/hosts` 파일 수정
    - IP 기반의 가상 호스팅
        - 하나의 아파치 웹 서버에서 물리적인 네트워크 카드에 여러 개의 IP 할당
        - 각 호스트들이 서로 다른 IP 주소를 이용할 수 있도록 하는 방식
        - `/etc/httpd/conf.d/` 안에 파일 작성
    - Port 기반 가상 호스팅
        - `/etc/httpd/conf.d/` 안에 파일 작성

## * 웹 서비스 실습

```powershell
<실습 전 기본 세팅>

yum -y install httpd

systemctl enable httpd --now

firewall-cmd --add-service=http --permanent

firewall-cmd --reload

nmcli con add con-name web ifname ens160 type ethernet ip4 192.168.100.150/24 gw4 192.168.100.2 +ipv4.addresses 192.168.100.110/24 +ipv4.addresses 192.168.100.120/24

nmcli con up web

가상 호스트

- vhost0

cd /etc/httpd/conf.d/

cp /usr/share/doc/httpd-core/httpd-vhost.conf 01-vhost.conf

vi 01-vhost.conf

<VirtualHost *:80>
    DocumentRoot /var/www/html1
    ServerName vhost1.est.main
    ServerAlias vhost1
</VirtualHost>

<Directory /var/www/html1>
    AllowOverride None
    Require all granted
</Directory>

:wq

mkdir /var/www/html1

echo vhost1 > /var/www/html1/index.html

--------------------------------------------

- vhost2

cd /etc/httpd/conf.d/

vi 02-vhost.conf

<VirtualHost *:80>
    DocumentRoot /var/www/html2
    ServerName vhost2.est.main
    ServerAlias vhost2
</VirtualHost>

<Directory /var/www/html2>
    AllowOverride None
    Require all granted
</Directory>

:wq

mkdir /var/www/html2

echo vhost2 > /var/www/html2/index.html

-----------------------------------------

vi /etc/hosts

192.168.100.150    servera.est.main  vhost1.est.main vhost1 vhost2.est.main vhost2

:wq

systemcl start httpd

firewall-cmd --add-service=http   --permanent 
firewall-cmd --reload

curl server.est.main
curl vhost0
curl vhost1
```
