## 📝 오늘 배운 내용 요약

1. 리눅스 명령어

```powershell
wc -l 줄
wc -w 단어
wc -m 문자
wc -c  바이트

cat : 파일 내용출력

more : 내용이 많은 파일을 화면 단위로 꾾어서 출력

head : 파일의 내용 중 처음부터 아래로 10줄 출력

tail : 파일의 내용 중 마지막부터 위로 10줄 출력

<디렉터리 및 파일 내용 변경 명령어>

mkdir : 디렉터리

touch : 0바이트에 파일을 만듬(파일)

cp : 파일 복사 및 붙여넣기

mv: 파일의 이동 및 이름 변경

rm : 파일 제거

grep '$6$' /etc/shadow  
fgrep '$6$' /etc/shadow
find . -name core -exec rm -rf {} \;
```
