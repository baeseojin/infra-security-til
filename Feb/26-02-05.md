    4. 리디렉션
        - < : 표준 입력 재지정문자
            - 키보드가 아닌 파일로 표준 입력을 지정하는 메타 문자
        - > : 표준 출력 재지정문자
            - 모니터가 아닌 파일로 표준 출력을 지정하는 메타 문자
            - > 덮어쓰기, >> 이어쓰기
            - 예제 코드
            
            ```powershell
            ls -lR / > ~/r01 2> ~/r02 : ~/r01에 표준 출력을 저장, ~/r02에 표준 에러 저장
            
            ls -lR / > ~/r03 2>&1 : 모든 출력값과 모든 에러 값을 전부 ~/r03에 덮어쓰기로 저장
            
            ls -lR / >> ~/r03 2>&1 : [동일한 파일의 기존 내용에 이어쓰기]
            ```

5. 퍼미션
    - root와 권한 소유자만 권한 부여 가능
    - 다중 사용자 접속에 필요
    
    <img width="855" height="435" alt="image" src="https://github.com/user-attachments/assets/d0908c3d-2a77-42ce-98e7-3e6e710e52b4" />

    
    - chmod 명령어
        - 예) `chmod u+r [파일이름]` : 소유자에게 읽기 권한 부여
        - 예) `chmod 777 [파일이름]` : 소유자, 그룹, 기타 사용자에게 모든 권한(읽기,쓰기,실행)을 부여
    
    <img width="813" height="407" alt="image" src="https://github.com/user-attachments/assets/be612c0e-0cea-49b5-960f-2a420e68e543" />

    
    - 소유권 설정
        
        <img width="688" height="193" alt="image" src="https://github.com/user-attachments/assets/9ebc0713-f3a9-493f-a389-3f7bafb56846" />

        
    - 기본 권한 설정
        - 파일을 만들 때 666 기본 권한
            - 파일의 기본 권한은 666 에서 umask 값과 뺀 값
        - 디렉터리를 만들 때 777 기본 권한
            - 디렉터리의 기본 권한은 777에서 umask 값을 뺀 값
        - root의 기본 umask는 022
        - 예) umask가 022이고 생성된 유형이 파일일 때 666에서 022를 뺀 값인 644가 기본 권한으로 설정
        - `umask [값]`으로 설정 가능
